Alt-Text.org

This project is currently paused.

Designed to help people share the alt text they write.

Seeking New Maintainer

The idea for a library of alt text has been around since the alt tag was added to the HTML spec. It's only recently that the technology available to the public has caught up. The project as it stands is a successful proof of concept. If you're interested in helping the project move forward, please contact admin@alt-text.org.

Main technologies used:

  • Node.js: API service
  • React: frontend UI
  • Google Cloud: hosting
  • Google Firestore: primarily storage of alt text
  • Google Cloud Storage: serving static frontend site
  • Pinecone.io: fuzzy matching of searched images
